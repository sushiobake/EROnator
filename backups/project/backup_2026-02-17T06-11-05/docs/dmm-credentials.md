## アフィリエイトID

### 通常のアフィリエイトID（リンク用）
- **ID**: `sok-001`
- **用途**: 通常のアフィリエイトリンク生成

### API用アフィリエイトID（APIリクエスト用）
- **ID**: `sok-990`（使用中）
- **その他**: `sok-991`～`sok-998`も発行済み（必要に応じて使用可能）
- **用途**: DMM Affiliate APIのリクエスト時に使用
- **確認方法**: DMM Affiliate管理画面 → Webサービス/API → アフィリエイトID（API用）

**重要**: 
- API登録時に末尾990～999のアフィリエイトIDが自動発行されます
- 通常のアフィリエイトID（`sok-001`）とは別のIDです
- APIリクエスト時は必ず末尾990～999のIDを使用してください
- 末尾が990～999以外ではエラーとなります

## 環境変数設定

`.env.local`ファイルに以下を設定：

```env
# DMM Affiliate API
DMM_API_ID=Rbb32LFV0FLeUMh5Bn3f
DMM_AFFILIATE_ID=sok-990  # API用アフィリエイトID（末尾990-999）
```

## 注意事項

1. **API IDとアフィリエイトIDは別物**
   - API ID: APIを呼び出すための認証情報
   - アフィリエイトID: アフィリエイト報酬を識別するID

2. **API用アフィリエイトIDの確認**
   - 管理画面の「Webサービス/API」ページで確認
   - 「APIID登録にともないアフィリエイトIDの末尾990～999が発行されました」というメッセージが表示されているはず

3. **セキュリティ**
   - API IDとアフィリエイトIDは機密情報です
   - `.env.local`は`.gitignore`に含まれているため、Gitにコミットされません
   - 他人に共有しないでください
