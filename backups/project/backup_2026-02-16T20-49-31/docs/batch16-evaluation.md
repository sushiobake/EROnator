## 良い点（GPT-5.1 の強み）

### 1. 致命的ミスが消えている
- **「ニット」** が一切出ていない。指示書の「ニット禁止」「タグ名は一覧と一致」が守れている。
- **「女性視点」** を根拠なく付けていない（本バッチ内に該当なし）。
- **additionalSTags** は all-tags の `s` に存在する表記のみ使用。創作・誤変換なし。

### 2. Sタグ（公式タグ）の使い方が適切
- 外国人、学生、パイズリ、快楽堕ち、近親相姦、アナル、シスター、同棲、少女、ハーレム、姉妹、淫乱、バイ など、本文・タイトルに基づいて additionalSTags に正しく入れている。
- 公式タグを「空のまま」にしている作品は、メモ上は理由がある場合（総集編のみなど）に限られている（※【8】は別問題）。

### 3. タイトル優先・主軸タグの拾い
- 人妻、サキュバス、異世界、ハーレム、教会、背徳、JK、オナホ、妹、総集編、人妻幼馴染 など、作品の核になる語をほぼ押さえている。
- 「物語」をメタ情報としてタグにしていない（作品4）など、付けてはいけない例も理解している。
- 「快楽」単体は避け「快楽堕ち」で表現（作品2）など、汎用語禁止も守れている。

### 4. characterName
- 神野晃、瑠衣 など、括弧で明記された名前を採用。作品タイトルを characterName に入れていない。

### 5. 自己認識
- 各作品で「指示書とのズレ・改善余地」を書いており、複合タグ重複・タグ0本・寝取り漏れを**自分で指摘**している。次の改善につなげやすい。

## 残っている課題（減点要因）

### 1. 【8】げーみんぐはーれむ — タグが 0 本（重大）
- **タイトルに「はーれむ」＝ハーレムが明記**されているのに、matchedTags も additionalSTags も空。
- 指示書「書いてある語を漏らさない」「主軸タグを漏らさない」に**明らかに反している**。
- メモでも「最優先で見直すべき」と書かれており、**提出前チェックで検出すべきだった**。
- → 1作品まるごとの見落としとして、減点は大きい。

### 2. 【10】人妻幼馴染 — 複合タグを 3 つ並べている
- **「人妻幼馴染」「幼馴染」「人妻」** の 3 つを維持。指示書冒頭・6・8の「複合タグとかぶりすぎ」で**3つ並べない**と明記されている。
- 「旧AIタグからの置き換えとして尊重」と理由は書いているが、**ルールは「3つ並べない」**なので未適用。
- → 指示書の重要ルールが 1 件で破れている。

### 3. 【1】【5】寝取り — 主軸タグの漏れ
- タイトルが「ワイルド式日本人妻の**寝取り方**」なのに、「寝取り／寝取られ」が matchedTags にない。
- 指示書「主軸タグを漏らさない」に照らすと不足。メモでも改善余地として挙がっている。

### 4. 【5】「バイ」Sタグ — 解釈のリスク
- 「バイト」と「バイ」（バイセクシュアル）の取り違えの可能性をメモが指摘。誤検出なら削除候補であり、提出前チェックで確認したい点。

### 5. 【6】【7】additionalSTags が空
- 【6】無人島の吉村さん 総集編: 総集編のみで他タグなし。シリーズ説明中心なら割り切りとも取れるが、不足気味の可能性。
- 【7】隣のJKに: JK のみ。タイトルに「脅されて」「入り浸られてます」とあり、本文次第では追加タグの余地あり。
