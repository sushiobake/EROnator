# 旧AIタグ 添削メモ

**このメモを読めば、タグ付けが実行できる。**

---

## 目的

1. **差別化**：その作品を他と区別して特定できるようにする
2. **ミスマッチ防止**：「〇〇に関する？」に Yes となる作品がヒットするようにする

---

## 必須の重点ルール（繰り返し指摘）

以下を守らないと40点止まり。90点を目指す。

| 項目 | 内容 |
|------|------|
| **作品を特徴づけるワード重視** | Aタグは「その作品だけの特徴」を表すワードを優先。全作品に共通するタグより、この作品を他と区別するワードを意識 |
| **タイトル優先** | タイトルに出ている語は**必ずチェック**する。合宿、オタクなど、差別化の核になりうる語を漏らさない |
| **書いてある語を漏らさない** | あらすじ・プレイ内容・キーワード・嗜好に列挙されているものは**まずそこから取る**。ラブホテル、合宿など明記されていれば必ず追加 |
| **推測禁止** | 「〜の可能性」程度の根拠では付けない。**明記されている場合のみ** |
| **用語の正確な理解** | 筆下ろし＝童貞の初体験 など、タグの意味を正しく把握してから付ける |
| **曖昧なら上位概念** | クラスメイト→高校は飛躍。中学・大学もありうるなら「学校」のほうが無難 |
| **不要なタグを入れない** | 作品の主軸・差別化にならないタグは付けない。調教など、周辺的要素は省く |
| **同義語は後処理** | 「水泳」と「プール」のような同義語は、後から統合スクリプトで調整。タグ付け時は思いついた方を付ける |

---

## タグ付けの手順

### ステップ1：読む

- タイトル、あらすじ、プレイ内容、キーワード、嗜好を把握
- 「＊プレイ内容」「キーワード」「この作品の嗜好」は**必ず確認**

### ステップ2：タグ候補を収集

**優先順位どおりに**：

1. **タイトル**：タイトルに出ている語を漏らさず候補にする
2. **あらすじ・プレイ内容・キーワード・嗜好**：列挙されているものを officialTagsCache / tagRanks と照らし合わせ候補にする
3. **差別化**：この作品の特徴を表す語が他にないか探す
4. **ミスマッチ防止**：「〇〇に関する？」に Yes になる要素がないか考える

**推測は最後**。書いてあるものから取ることを優先する。

### ステップ3：確定

- matchedTags / suggestedTags / additionalSTags / characterName を決定
- 公式タグにあれば additionalSTags、なければ matchedTags または suggestedTags
- 下記「ルール」を確認し、該当するものを適用・除外

### ステップ4：確認（必須）

- **タイトル＋作品コメント** と **付けたタグ** が矛盾していないか
- **漏れ**：書いてあるのに付けていないものがないか
- **過剰**：否定されている、推測のみ、主軸でないタグは削除

---

## ルール（該当したら適用）

| ルール | 内容 |
|--------|------|
| 推測で付けない | 初恋、人妻、筆下ろしなど、あらすじに明確な記載がなければ付けない。「可能性」は不可 |
| 類似語の区別 | 借金≠金欠、セックスレス≠妊活。用語の意味を正しく使う |
| 過大解釈 | 盗撮シチュがあるだけ→盗撮タグ×。キーワードを優先 |
| 曖昧なタグ | 彼女あり、元カノなど、分かりにくい・広義すぎるタグは付けない。「学生」と「女学生」は重複のため「学生」のみ使用 |
| 上位概念 | クラスメイト→高校×。中学・大学もありうるなら「学校」 |
| 総集編 | 収録作に明記されていれば該当タグを追加 |
| 用語の正確さ | 催●は「催眠」、眠姦は存在しない（正しくは「睡眠姦」）。造語や誤った用語は避け、正式な用語を使用 |
| 嗜好ブロック | 「この作品の嗜好」に列挙されているものは必ず見る |
| 特徴ある語 | 性奴隷、チート、種付け、合宿、オタク、おじさん など、タイトル・内容に明記されていればタグにする |

---

## キャラタグ（characterName）

- 明らかにある場合のみ。1人でOK
- 主要キャラ（翠音など看板キャラ）は漏らさない

---

## 出力形式

- matchedTags, suggestedTags, additionalSTags, characterName を含む
- tagReasoning：各タグの理由を記載（ミス防止のため）
