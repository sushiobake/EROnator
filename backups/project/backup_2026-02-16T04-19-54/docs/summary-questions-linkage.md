# まとめ質問とタグの連動

## 方針

- まとめ質問の `displayNames` は **実際に存在するタグ（公式タグ）と 1:1 で対応**させる。
- 「余計なタグ」（DB/公式に存在しない名前）は入れない。
- 正として参照したのは `config/officialTagsCache.json` の `tags` 配列（公式タグの displayName）。

## 今後の運用

- まとめを追加・変更するときは、必ず `officialTagsCache.json` または実際の Tag テーブルと照合し、**存在する displayName だけ**を `displayNames` に書く。
- 新規まとめは「既存タグからテーマでまとめる」形で設計する。

## 注意

- 上記タグ名はすべて **公式タグ一覧に存在するもの** に限定している。
- 採用する場合は、必ず `officialTagsCache.json` で表記を再確認し、`displayNames` は **そのまま** 使うこと。
- まとめを追加するときは、`config/summaryQuestions.json` の `summaryQuestions` に 1 件追加し、`id`, `label`, `questionText`, `displayNames`（と必要なら `erotic`）を、**既存タグと連動するように**書く。
